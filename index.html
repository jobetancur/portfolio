Acerca de mí

Desarrollador Full Stack en formación con conocimientos en HTML, CSS y JavaScript. Soy una persona que le gusta reinventarse constantemente en busca de alcanzar un mejor desempeño. Mi perfil está orientado al logro, la proactividad, la resolución de problemas y un espíritu de aprendizaje constante. He encontrado una gran pasión por el desarrollo web y la implementación de aplicaciones que busquen suplir las necesidades sociales. Busco oportunidades que me permitan aprender cada día a través de retos, trabajar en equipo e implementar mis conocimientos para el bien común.

Mis habilidades

HTML 5

Lenguaje de Marcas de Hipertexto. Es utilizado para estructurar de la manera más básica, la forma en la que se desea mostrar la información consignada dentro de una página web. HTML no es un lenguaje de programación, sino, una serie de herramientas que ayudarán a que el contenido se vea y se comporte de una manera determinada.

CSS 3

Hojas de estilo en cascada o CSS derivado del ingles (Cascading Style Sheets). Es usado para entregar representación gráfica o renderización al trabajo realizado con HTML. Es adecuado para ubicar elementos, dar color a texto o animar elementos dentro de una página web, entre muchas funcionalidades más.

JavaScript

Mejor conocido de manera abreviada como JS, es un lenguaje de programación ligero y enfocado principalmente a páginas web y que, ayuda fundamentalmente a agregarle funcionalidad a las mismas. Sin embargo, gracias a si versatilidad este lenguaje puede ser utilizada en entornos que no son de navegador.

React

Es una de las librerías para JavaScript más populares del momento. Esta es de uso open source y ayuda principalmente al desarrollo de interfaces de usuario en casi cualquier escala de complejidad. Actualmente compañías de primer nivel tales como Facebook, Netflix y Uber hacen uso de esta librería.

Git

Es un controlador de versiones que hoy en día es probablemente el más utilizado por el entorno del desarrollo web. GIT hace que trabajar en equipo sea mucho más fácil al permitir clonar de manera local el repositorio en desarrollo. Por otro lado, también nos permite realizar una trazabilidad precisa a cada uno de los cambios realizados por el equipo.

GitHub

Es el equivalente a Google Drive pero del mundo del desarrollo. Nos ayuda a trabajar en equipo y asociar nuestros avances realizados de manera local con ayuda de GIT. Este sistema de almacenamiento también es ideal para publicar proyectos personales para darlos a conocer a la comunidad.
